---------------------------------------------------------------------------------------------------------------------------
- Objectif 
	Montrer les fonctionnalités de la carte EasyDriver 4.4 pour contrôler un moteur pas à pas.

***************************************************************************************************************************
- Documentation dans le répertoire "Ressources"
	* Circuit A3967.pdf
	* Interface de puissance : EasyDriver_v44_sch.pdf
	* Caractéristiques du moteur pas à pas ITC-CNC-2 :  N = 200 pas - U = 12V - C = 200g/cm 
	* Shield Arduino miniE_arduino_shield_v1-3
